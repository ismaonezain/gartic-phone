# Gartic Phone

## Description
Gartic Phone is an engaging online drawing and guessing game, where players take turns creating drawings and guessing them based on previous images. Suitable for players of all ages, it combines creativity and fun with family or friends! 

## Installation
To set up the project on your local machine, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/ismaonezain/gartic-phone.git
   ```
2. Navigate into the project's directory:
   ```bash
   cd gartic-phone
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, use the following command:
```bash
npm start
```
You can then open your web browser and navigate to `http://localhost:3000` to play the game.
